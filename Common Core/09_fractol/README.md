fract'ol
Description

fract'ol is a basic computer graphics project where the goal is to create an interactive fractal explorer using the MiniLibX library. The program allows users to explore the Mandelbrot and Julia sets, zoom in/out with the mouse, move the viewport using arrow keys, and visualize the fractals with smooth color gradients. The project introduces concepts of complex numbers, optimization in computer graphics, and event handling.

Instructions

Clone the repository:

git clone <your-repo-url>
cd fractol

Compile the project:

make

Run the program:

./fractol mandelbrot
./fractol julia <real_part> <imaginary_part>

mandelbrot: opens the Mandelbrot fractal.

julia <re> <im>: opens a Julia fractal with custom complex parameter.

Controls:

Mouse scroll: zoom in/out at the mouse position.

Arrow keys: move the viewport.

ESC key or clicking the window close button: exit the program.
